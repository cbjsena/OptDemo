1. 데이터 세트 개요

LINER-LIB 2012는 정기선 해운 네트워크 설계(LSND) 문제를 위한 표준 벤치마크 데이터 세트로, 실제 글로벌 컨테이너 해운 데이터를 기반으로 구성되었습니다. 7개의 서로 다른 규모와 특성을 가진 인스턴스를 제공하여 다양한 알고리즘과 접근법의 성능을 비교할 수 있습니다.

3. 데이터 구조 및 파일 형식

데이터 구성 요소

모든 데이터는 CSV 형식으로 제공되며, 다음과 같은 파일들로 구성됩니다:

📍 지리적 데이터
  • ports.csv - 모든 항구 정보 (통합)
  • dist_dense.csv - 항구간 거리 (밀집형)
  • dist_sparse.csv - 항구간 거리 (희소형)
🚢 운영 데이터
  • fleet_data.csv - 선박 클래스 정보
  • fleet_{instance}.csv - 인스턴스별 선단
  • Demand_{instance}.csv - 인스턴스별 수요
  • rots.json - 경로 정보 (JSON)

데이터 활용 방법
연구 및 개발
  • 알고리즘 벤치마킹: 새로운 LSND 알고리즘의 성능 평가
  • 성능 비교: 기존 연구 결과와의 객관적 비교
  • 확장성 테스트: 다양한 규모에서의 알고리즘 성능 검증
  • 모델 검증: 수리적 모델의 정확성 확인
교육 및 학습
  • 학부/대학원 교육: 실제 데이터를 활용한 실습
  • 최적화 소프트웨어 학습: Gurobi, CPLEX 등 솔버 활용
  • 산업 이해: 실제 해운업계 구조와 문제 파악
권장 시작 순서
  1. Baltic - 기본 알고리즘 구현 및 검증
  2. WAF - 허브-스포크 구조 이해
  3. Mediterranean - 중간 규모 문제 해결
  4. World Small - 글로벌 네트워크 이해
  5. Pacific, Asia-Europe, World Large - 대규모 문제 도전
