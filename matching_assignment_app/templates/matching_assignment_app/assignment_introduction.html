작업 배정 문제 (Assignment Problem)

범주: Ⅱ. 매칭 및 할당 (Matching & Assignment)

작업 배정 문제는 조합 최적화(Combinatorial Optimization) 분야의 가장 대표적이고 기본적인 문제 중 하나입니다. 이 문제는 동일한 수의 작업(Jobs)과 작업자(Workers)가 있을 때, 각 작업자-작업 쌍에 대한 비용 또는 시간이 주어졌을 때, 전체 비용(또는 시간)을 최소화하도록 각 작업자에게 하나의 작업만을 할당하는 방법을 찾는 것을 목표로 합니다.

1. 문제 설명 및 핵심 제약

작업 배정 문제는 다음과 같은 간단하지만 강력한 규칙을 따릅니다:

  • 모든 작업자는 단 하나의 작업에만 배정될 수 있습니다.
  • 모든 작업은 단 한 명의 작업자에게만 배정되어야 합니다.
  • 목표는 모든 배정으로 인해 발생하는 비용의 총합을 최소화하는 것입니다. (또는 이익을 최대화할 수도 있습니다.)

이러한 1:1 매칭 제약 조건은 다양한 현실 문제에 적용될 수 있는 모델의 기초가 됩니다.

2. 산업 응용 분야

작업 배정 문제는 여러 산업 분야에서 효율적인 자원 할당을 위해 사용됩니다.

  • 인력 관리: 특정 기술을 가진 직원들을 가장 적합한 프로젝트나 태스크에 배정하여 전체 프로젝트 완료 시간을 단축합니다.
  • 생산 공정: 여러 개의 기계를 각각 다른 생산 공정에 할당하여 총생산 비용을 최소화합니다.
  • 교통 및 물류: 여러 대의 배송 트럭을 각각 다른 배송 지역에 할당하여 총 운행 거리를 최소화합니다. (VRP의 단순화된 형태로 볼 수 있습니다.)
  • 컨설팅 및 서비스: 전문 분야가 다른 컨설턴트들을 각각의 고객사 또는 문제에 배정하여 고객 만족도나 프로젝트 성공률을 최대화합니다.

3. 수학적 모델

결정 변수:
$$ x_{ij} = \begin{cases} 1 & \text{if worker } i \text{ is assigned to job } j \\ 0 & \text{otherwise} \end{cases} $$
목적 함수: 비용 최소화

$$\min \sum_{i=1}^{n} \sum_{j=1}^{n} c_{ij} x_{ij}$$

$$ \text{where } c_{ij} \text{ is the cost when resource } i \text{ is assigned to project } j $$

제약 조건:
  • 각 작업자는 정확히 하나의 작업에 배정되어야 합니다
    $$\sum_{j=1}^{n} x_{ij} = 1 \quad \forall i = 1, \dots, n$$
  • 각 작업은 정확히 한 명의 작업자에게 배정되어야 합니다
    $$\sum_{i=1}^{n} x_{ij} = 1 \quad \forall j = 1, \dots, n$$

이 문제는 헝가리안 알고리즘(Hungarian Algorithm)과 같은 매우 효율적인 전문 알고리즘으로 해결할 수 있습니다.
